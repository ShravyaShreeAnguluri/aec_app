1. Create rooms first
   - CLASSROOM rooms
   - LAB rooms

2. Create sections for all years and categories
   Example:
   - 2nd year NON_THUB sections -> working_days = 0,1,2,3,4
   - 2nd year THUB sections -> working_days = 0,1,2,3,4,5
   - 3rd year sections -> working_days = 0,1,2,3,4,5
   - lunch_after_period = 2 or 3 depending on that year/category
   - thub_mode = MORNING_ONLY only for THUB sections

3. Create subjects
   THEORY:
   - weekly hours correctly entered
   - allow_same_day_repeat only if needed

   LAB:
   - is_lab = true
   - min_continuous_periods / max_continuous_periods
   Example:
   - THUB labs -> 2
   - NON_THUB labs -> 3
   - special NON_THUB labs -> 4

   ACTIVITIES like FIP / PSA:
   - create as subjects
   - do not hardcode in code
   - if fixed, mark is_fixed=true and set fixed_day/fixed_days/fixed_start_period/fixed_span

4. Map faculty to subjects
   - primary faculty first
   - daily/weekly max hours if needed

5. Generate timetable department-wise and academic-year-wise
